# VegaPolarisTheme
I got sick of the new ServiceNow dark mode theme that is still pretty hard on the eyes so I cooked up my own. Its much more muted and is designed to be added as a variant of the core polaris themes for easy installation. Its named after Vega one of the brighest stars in the sky.
